## 一、研究目的

用数据回答一个问题:**原版海龟交易法在 A 股个股上能不能赚钱?**

SPEC 要求:不改逻辑、不加过滤、不做优化,纯粹验证原版海龟规则(20日突破入场 / 2ATR止损 / 10日离场 / 单笔风险1%)在沪深300成分股 2022-2025 的基线表现。在此基础上,诚哥追加两轮优化(大盘择时、双均线+量能)探索天花板。

## 二、两个关键决策(诚哥拍板)

### 决策1:数据源 SPEC 写 mootdx → 改 astock 前复权

SPEC 指定 mootdx 取数。实测发现:
- mootdx 0.11.7 日K**默认不复权**,前复权需自己用除权除息因子序列实现
- 自实现前复权有 **look-ahead 风险**(用未来复权因子调整历史价格,易出 bug)
- mootdx 与"现有数据工具链"并不一致(项目主数据源是 astock)

astock parquet(`E:/astock/daily/stock_daily.parquet`)已内置 `adj_factor`,前复权一行实现,无 look-ahead 风险:
- 1432万行 / 5793只 / 2009-01~2026-06 / 沪深300全覆盖 / adj_factor 非空率100%
- 成分股用 akshare `ak.index_stock_consindex(symbol="000300")` 取(300只,0.3秒)
- 2只 astock 无数据(001280.SZ / 600930.SH,新进成分)排除,实际回测 298 只

### 决策2:仓位公式 SPEC 笔误 → 改标准海龟 1% 风险

SPEC §2 仓位公式:`买入股数 = 单笔风险 / (2 × ATR / 当前价 × 100)`

**数学推导坐实是笔误**:
```
shares = 单笔风险 / (2×ATR/价×100)
=> 2×ATR×shares = 单笔风险 × 价 × 100 / (2×ATR) × 2×ATR ... 化简
=> 2×ATR×shares = 100 × 价          (与 ATR 无关!)
=> 单笔风险 = 100 × 价
=> 单笔风险占比 = 价 / 10000
```
沪深300均价~31元 → 单笔风险 0.31%,远非 SPEC 声称的 1%。v1 实测 avg_risk_pct=0.31% 完全吻合。

**改标准海龟**:`shares = 单笔风险 / (2 × ATR)`,单笔风险固定=1%账户(与 ATR 反比,与价格无关)。v2 实测 avg_risk_pct=0.893%≈1%(被20%单票上限轻微截断)。

**这是修 SPEC 笔误,不是改策略逻辑。** 教训:仓位/风险公式要数学验证(代入典型值看单笔风险是否=声称值)。

### 基线(标准海龟,commit 191535a)
原版海龟,A股最小适配(收盘价突破、次日开盘执行T+1)。**年化0.5%,保本微赚**。胜率34%低但盈亏比2.41(典型趋势跟踪:小亏多次+大盈少次)。100%跑赢沪深300是因基准跌20%、海龟止损2ATR单笔亏损有限所以不亏20% - **不是海龟强**。

### v5(+MA20大盘择时)
入场额外要求沪深300收盘>沪深300MA20(大盘多头)。**全面小幅改善**:年化0.67%、胜率36.3%、盈亏比2.57、正收益61.7%、回撤23.0%。交易减少25%(避开熊市假突破)。**方向对但幅度有限**。

### v6(双均线+量能,失败)
入场要求双均线多头(沪深300>MA20 AND MA20>MA60)+ 量比≥1.5。**效果反而变差**:年化-0.14%(转亏)、胜率29.8%、正收益34.7%。只剩1270笔(基线23%)。

**v6 失败原因**:
1. **过滤过严**:双均线+量能只剩23%信号,砍掉真突破
2. **高位追涨反效果**:双均线多头意味趋势已走一段(滞后),此时放量突破(量比≥1.5)多是**趋势末端顶部放量/诱多**,胜率反降
3. **量能在多头高位是反向指标**:放量突破在低位是启动,在高位是见顶。v6 = 高位+放量 = 高位追涨

## 四、结论

### 原版海龟在 A 股(沪深300 / 2022-2025)不能赚钱

- **基线年化0.5%**,扣无风险利率(约2-3%)后为负
- **v5择时天花板0.67%**,仍扣利率为负
- **v6加强择时反效果**,证明择时不是越严越好
- 100%跑赢沪深300是因基准跌20%(**熊市**),不是海龟强;若遇牛市基准涨,海龟大概率跑输(趋势跟踪滞后)

### 为什么海龟在 A 股不行

1. **趋势跟踪固有低胜率**:34%胜率是趋势跟踪本质(小亏多次+大盈少次),A股散户多、震荡市多,假突破频繁
2. **T+1 约束**:次日开盘执行,隔夜跳空吃利润(参考 [[tail-30min-strategy-falsified]]:强势股T+1平均低开0.7%)
3. **2022-2025熊市长**:沪深300跌20%,反弹段短且急,海龟突破入场常买在反弹末端
4. **追涨天然吃亏**:海龟突破=追涨,A股追涨隔夜方向错(75.6%概率低开)

### 择时有效但天花板低

- 单MA20择时(v5)有效:避开熊市,指标全改善
- 但加强择时(v6双均线+量能)过严反效果
- 择时能从0.5%提到0.67%,无法跨越"扣利率为负"的门槛

## 五、教训沉淀

1. **数据源前复权**:优先用 astock(adj_factor 内置),避免 mootdx 自实现前复权的 look-ahead 风险。前复权基准日用数据末日或回测期末日(同比例缩放不影响信号)。
2. **仓位公式数学验证**:代入典型值(ATR=1,价=10)看单笔风险是否=声称值。SPEC `2*atr/price*100` 代入得风险=100×价(与atr无关),0.31%≠1% - 笔误。
3. **SPEC 不自洽要 flag**:成本表格(滑点0.1%)与汇总0.0351%/0.1351%(隐含0.01%)不自洽,按表格明细实现并在 report 标注。
4. **择时不是越严越好**:单MA20有效,双均线+量能过严反效果。择时过滤要测边际收益,非越强越好。
5. **量能是双刃剑**:放量突破在低位是启动信号,在多头高位(双均线多头)是见顶信号。量能过滤不能脱离位置盲用。
6. **趋势跟踪+熊市+T+1 三重压制**:A股趋势跟踪策略天然吃亏,低胜率是固有非bug。海龟类策略在A股需配合更强择时(可能仍不够)或转向均值回归/波段。
