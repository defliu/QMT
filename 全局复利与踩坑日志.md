## 二、已沉淀的关键教训

### 1. 初选 + 精筛不能机械写成同日 AND

来源：黄氏主升浪策略。

错误写法：

```text
combo_XG = box_breakout_XG AND double_zhongjun_XG
```

问题：箱体突破要求均线黏连、压缩末端；双中军要求多头排列、均线发散、动能确认。二者是先后阶段，不是同日并列过滤。

正确写法：

```text
box_window_hit = 最近 N 日出现 box_breakout_XG
combo_XG = box_window_hit AND 今日 double_zhongjun_XG
```

### 2. 形态看起来“最完整”不等于有正 alpha

来源：黄氏双中军 3 年回测。

双中军因覆盖趋势 + 动能 + 突破 + 大盘环境被评为 Top 1，但 2023-2026 中小盘回测显示信号胜率约 34%，完整工程持仓胜率约 30.7%，累计收益大幅为负。后续策略评审必须把“形态完整性”和“统计正期望”分开。

### 3. QMT_STRATEGIES 是量化研发落地区，也承担项目导航入口

来源：工作区治理。

`D:\QMT_STRATEGIES` 主要服务 CC/MIMO/RS 的研发、SPEC、代码、回测执行，同时保留 `全局控制台.md` 与 `Project_*` 作为项目导航入口。`D:\Workspace` 仅作为历史过渡区/旧控制台镜像，避免双入口长期并存。

### 4. 知识库整理要生成可用版 MD，不是原文搬运

来源：通达信/QMT 资料整理。

原始 PDF/DOC 放 `00_Inbox`，抽取文本可放 `_extracted_*`，长期使用的知识要整理到 `60_工程知识库`，并做速查表、复刻注意事项、CC 检查清单。

### 5. 任务超过 10 分钟必须入板

来源：并行任务管理。

没有任务卡就不开工；没有交付物路径不算完成；没有 Hermes/诚哥验收不进 DONE。

### 6. QMT 内置回测外部 Python 路径不能当研究端撮合引擎

来源：`Project_05_QMT内置回测系统可行性`。

`stgentry.run_file()` 能跑生命周期和行情，但 `passorder` 不产生 `deal/order/account` callback，callback cache 为空。后续研究端回测继续走自建回测工厂；QMT GUI 回测只能作为人工辅助验证。
