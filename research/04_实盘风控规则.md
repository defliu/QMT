# 实盘风控规则

用于沉淀 QMT 实盘/模拟端运行时的风控约束。

## 风控层级

| 层级 | 内容 |
|---|---|
| 市场风控 | 大盘环境、市场情绪、整体仓位上限 |
| 股票风控 | ST、停牌、跌停、黑名单、财务风险 |
| 交易风控 | 单票仓位、委托失败、重复下单、成交检查 |
| 策略风控 | 止损、止盈、时间止损、评分下降、换仓 |
| 系统风控 | QMT连接、xtquant异常、数据缺失、日志告警 |

## 实盘前检查

```text
1. QMT 端策略文件编码是否 GBK
2. validate_qmt_file.py 是否 ALL PASS
3. 是否排除 context_mock.py
4. passorder 返回值是否正确判断
5. 是否处理涨跌停/停牌/ST
6. 是否有重复下单保护
7. 是否有当日操作记录
8. 是否有日志和异常告警
9. 是否在测试账号验证
10. 回测结果是否符合预期
```
