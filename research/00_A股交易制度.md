# A股交易制度与实盘约束

## 核心规则

| 规则 | 对策略的影响 |
|---|---|
| T+1 | 当日买入不能当日卖出，止损/换仓逻辑必须考虑不可卖状态 |
| 涨跌停 | 信号出现不等于可成交，回测必须处理无法买入/卖出 |
| 集合竞价 | 09:15-09:25 价格机制特殊，信号需要单独设计 |
| 午休断档 | 11:30-13:00 无连续交易，日内策略需处理时间断层 |
| 停牌/复牌 | 数据缺口、不可交易、复牌跳空风险 |
| ST/退市风险 | 股票池过滤和风控黑名单必须处理 |
| 新股/次新股 | 波动、涨跌停、流动性和数据长度特殊 |
