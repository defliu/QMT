# 诚哥量化策略研究知识库

本目录用于沉淀 Hermes + 多Agent 研究流程的长期成果。

## 使用原则

- Hermes 主Agent负责和诚哥讨论、调度临时研究Agent、汇总结论。
- 临时 Kimi Research Agent 负责资料研究、A股背景、研报/文章归纳。
- 临时 DeepSeek Strategy Agent 负责策略反证、风险审查、回测设计质疑。
- CC 负责代码实现、回测、验证，Hermes 不直接写代码。
- 所有策略结论必须经过回测结果验证，不以模型判断作为最终依据。

## 推荐流程

```text
诚哥提出策略想法
  ↓
Hermes 拆解研究问题
  ↓
Kimi Research Agent：查资料/背景/类似策略
  ↓
DeepSeek Strategy Agent：反证/风险/回测设计
  ↓
Hermes 汇总为策略卡片或 SPEC
  ↓
CC 执行回测/开发
  ↓
回测结果反哺知识库
```

## 目录说明

| 文件/目录 | 用途 |

| 02_策略模型库.md | 策略结构、入场/出场/仓位框架 |
| 03_回测坑位.md | 未来函数、滑点、涨跌停、停牌等问题 |
| 04_实盘风控规则.md | 实盘下单、仓位、黑名单、异常保护 |

| paper_notes/ | 研报/文章/资料笔记 |
| specs_drafts/ | 研究后形成的 SPEC 草稿 |
