# 回测坑位

回测是策略正确性的核心验证，不能只看模型推理。

## 必查问题

| 坑位 | 说明 | 检查方式 |
|---|---|---|
| 未来函数 | 使用了当时不可得的数据 | 检查数据时间戳、财报发布日期、成分股日期 |
| 生存者偏差 | 只用当前仍存在股票 | 使用历史股票池/退市股数据 |
| 涨跌停成交 | 信号出现但无法成交 | 回测加入涨停买不进、跌停卖不出约束 |
| 停牌处理 | 停牌期间价格/成交异常 | 停牌日不可交易，数据不乱补 |
| 滑点手续费 | 纸面收益虚高 | 加入手续费、印花税、滑点 |
| 流动性 | 小票容量不足 | 限制成交金额占比/换手 |
| 复权错误 | 前后复权混用 | 明确复权口径 |
| 参数过拟合 | 调参适配历史 | 样本外测试、滚动测试 |
| 单笔贡献过大 | 收益靠少数运气交易 | 看收益分布和剔除极端值后表现 |
| 换手过高 | 成本吞噬收益 | 检查年化换手和交易次数 |
