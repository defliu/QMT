### P0-2：加入交易成本模型
| 项 | 详情 |
|----|------|
| **问题** | 月均换手100%，完全未扣除佣金、印花税、冲击成本 |

**成本参数（A股小盘股保守估计）**：
```
印花税: 0.1%（仅卖出）
佣金: 0.03%（双边）
冲击成本: 0.2%（双边）
买入总成本: 0.23%
卖出总成本: 0.33%

```

## 二、P1 核心优化（共4项，P0完成后执行）

### P1-1：因子权重改ICIR加权
| 项 | 详情 |
|----|------|
| **问题** | 当前权重（30/25/25/20）拍脑袋，ROE的ICIR仅0.16却占20%权重 |
| **修改文件** | `scoring.py` |
| **修改位置** | `scoring.py` 第26-31行FACTOR_WEIGHTS |
| **验证标准** | 单因子权重与ICIR绝对值正相关 |
| **预期影响** | IC提升5-10%，年化提升0.5-1pp |

**目标权重（基于现有ICIR）**：
```
BP: 0.46 → 28%
反转1月: 0.52 → 31%
低波60d: 0.49 → 29%
ROE: 0.16 → 12%
```

### P1-2：换手率缓冲机制
| 项 | 详情 |
|----|------|
| **问题** | 月均换手100%，每只股票平均持有仅1个月，产生巨额交易成本 |
| **修改文件** | `backtest.py` |
| **修改位置** | `backtest.py` 第88行选股逻辑 |
| **验证标准** | 月均换手从100%降至50-60% |
| **预期影响** | 年化成本节省2-4pp |

**缓冲规则**：已持仓股票新评分仍在前`top_n × 2`名则保留，不强制换仓。

### P1-3：修正行业中性化错误结论
| 项 | 详情 |
|----|------|
| **问题** | `KNOWLEDGE.md` 误称"行业中性化效果不显著（已自然分散）" |
| **修改文件** | `KNOWLEDGE.md` + `HERMES_REVIEW.md` + `backtest.py` HTML模板 |
| **修改位置** | `KNOWLEDGE.md` 第51-53行 |
| **验证标准** | 文档结论与`industry_neutralize_compare.csv`数据一致 |
| **预期影响** | 无收益影响，仅修正结论 |

**正确结论**：
```
TOP10 +行业中性化: 4.7% → 6.0% ✅ +1.3pp
TOP20 +行业中性化: 6.5% → 5.3% ❌ -1.2pp
TOP30 +行业中性化: 6.8% → 6.6% 轻微负贡献
→ 仅TOP10持仓时启用行业中性化；TOP20/30不启用
```
