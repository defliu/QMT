## 策略框架

- **选股范围**: 中证500 + 中证1000（市值排名 301~1800，动态滚动避免生存偏差）
- **因子组合**: BP(30%) + 反转-1月(25%) + 低波60d(25%) + ROE(20%)
 - **调仓方式**: 双月调仓 + 次日收盘价成交 + 等权持仓
 - **评分方法**: 因子截面 z-score → winsorize(1%/99%) → 加权求和 → 0~100 归一化
 - **交易成本**: 单边 0.2%（佣金+印花税+滑点）
 - **财报处理**: 考虑披露滞后45天

 ## P0 修正内容

 | 缺陷 | 影响 | 修复 |
 |------|------|------|
 | 静态Universe（生存偏差） | 2-4%年化高估 | 动态滚动universe，每调仓日按当时市值排名过滤 |

 | 收盘价成交前视 | 0.5-1%年化高估 | 评分日次日收盘价成交 |

## 10万本金实盘参数回测（2026-07-19）

> 路径A参数优化 + P0验证后，针对小资金实盘的专项回测。

### 核心结论
- **小市值(0-30亿)是最大alpha**：全池仅2.2%，0-30亿达10-15%
- **容量瓶颈是上限非门槛**：10万~2000万都能拿满收益，>2000万才衰减
- **10万本金反而更优**：冲击成本极低，年化(17.1%)高于大资金基线(15.5%)

### 推荐实盘参数
| 参数 | 取值 |
|------|------|
| 市值区间 | 0-30亿 |
| 调仓频率 | 双月(2M) |
| 持仓数 | TOP80 |
| 止损线 | -12% |
| 成交额阈值 | 2000万（过滤低流动性垃圾股） |
| 交易成本 | 0.15%单边（小资金实际可拿） |
| 单票上限 | 2% |

### 回测结果（全周期 2018.07-2026.06）
| 指标 | 阈值2000万 | 阈值1000万 |
|------|-----------|-----------|
| 年化收益 | **17.1%** | 14.5% |
| 夏普比率 | **0.55** | 0.49 |
| 最大回撤 | **-19.8%** | -23.0% |
| 总收益(8年) | **158.5%** | 125.6% |
| 10万终值 | **25.8万** | 22.6万 |

### 阈值对比
- 2000万 > 1000万：过滤超低流动性小票后组合质量提升，回撤更窄、收益更高
- 1000万放进了更多极低流动性票，实际冲击成本更高、极端波动更大

### 实盘预期
- 中性：12-15%年化（考虑小市值滑点略高于假设）
- 10万本金8年预期终值：25-28万

脚本：`research/multi_factor_ic/backtest_10w.py` / `backtest_10w_1000w.py`

### 盘中止损+替换（v2 修正版）

| 方案 | 年化 | 夏普 | 最大回撤 | 止损次数 | 验收 |
|------|------|------|---------|---------|------|
| 双月无止损 | 1.8% | 0.07 | -24.1% | 0 | — |
| 止损-8% | 3.5% | -0.03 | -23.0% | 379 | ✅ <15% |
| **止损-12%** | **3.6%** | **0.00** | **-22.4%** | 156 | ✅ |
| 止损-16% | 3.5% | 0.01 | -22.5% | 61 | ✅ |

v2 修正要点：
- 前日收盘判断 → 次日收盘执行（消除前视）
- 现金结算：每仓本位独立跟踪，不搞复利因子传递
- 替换排除同票（`cand != code`），避免自我替换
- 止损年化均 < 15%，次数均 < 570（调仓57×持仓20×0.5）

> ⚠️ v1版"93.7%年化收益"为回测Bug（前视偏差+复利因子错误+自我替换），v2已全面修复。

机制：持仓票前一日收盘跌幅超止损线→当日收盘卖出→以当日收盘买入评分最高未持仓票替代。

实现函数：`backtest_stop_loss()` / `compare_stop_loss()`

明细见 `reports/stop_loss_comparison.csv`
