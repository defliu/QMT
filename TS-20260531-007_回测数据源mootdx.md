#### mootdx 数据接口

```python
from mootdx.quotes import Quotes

client = Quotes.factory(market='std')
# symbol: 6位纯代码（如 '600519'，不需后缀）
# category: 4=日线, 5=周线, 6=月线
# offset: 返回多少根K线（建议设较大值如 800 确保覆盖回测周期）
df = client.bars(symbol='600519', category=4, offset=800)

# df 字段: open, close, high, low, vol, amount, datetime
# datetime 是 int 如 20240530
```

**注意：** mootdx 的 `symbol` 参数只用 6 位纯数字代码，不能带 `.SH` `.SZ` 后缀。需在函数内部做转换：
- `code.endswith('.SH')` → 取前6位
- `code.endswith('.SZ')` → 取前6位
- 纯6位不变

mootdx 的 market 参数：0=深圳, 1=上海，根据 code 前缀判断：
- `6`开头 → `market=1`
- `0/3/002/300`开头 → `market=0`
- 但 `Quotes.factory(market='std')` 会自动处理，直接用 std 即可
