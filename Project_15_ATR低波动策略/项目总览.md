## 策略逻辑

### 选股条件（S010 低波动+换手策略 重构版）

| 条件 | 参数 | 来源 |
|------|------|------|
| ATR%% < 6%% | `atr_threshold: 6.0` | 5年29因子归因最强(+0.313%%) |
| 换手率 1-8%% | `min_turnover: 1.0` / `max_turnover: 8.0` | 5年归因第二(+0.158%%) |
| 成交额排序 top N | `max_hold: 3` | 流动性过滤 |

### 仓位控制

| 参数 | 值 | 说明 |
|------|-----|------|
| 本金 | 10万元 | `capital_base: 100000` |
| 最大持仓 | 3只 | `max_hold: 3` |
| 单只仓位 | 30% | 每只固定总资产的30% |
| 总仓位上限 | 90% | `max_total_ratio: 0.90` |
| 排除其他策略持仓 | ✅ | 选股时读取账户全量持仓，排除 |

### 退出机制

| 类型 | 参数 | 说明 |
|------|------|------|
| 止损 | -8% | 硬止损 |
| 止盈 | +20% | 固定止盈 |
| 移动止盈 | -10% from peak | 从持仓最高点回落10% |
| 条件失效退出 | ✅ | ATR或换手率不满足条件时卖出 |

### 买入窗口
- 生产版: 14:50-15:00（尾盘）
- 全天调试版: 全天可交易（`--atr-allday` 构建时硬编码 `_DEBUG_MODE = True`）

## 与6+2代码的关系

- **完全独立**: 源文件在 `atr_lowvol/` 目录，不引用 `core/`、`adapters/`、`config/global_config.yaml`
- **运行时文件隔离**: 使用 `atr_` 前缀（`atr_holdings.json`、`atr_trade_log.csv`），不与旧策略冲突
- **构建隔离**: `--atr` 参数走独立构建路径，不走旧SOURCE_FILES
