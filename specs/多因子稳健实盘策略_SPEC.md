# 多因子稳健实盘策略 — SPEC

> 版本: v1 (稳健可实盘型)
> 日期: 2026-07-20
> 路线: 承接 2026-07-03/04 多因子 harness 研究成果

## 0. 前置结论(为什么是这套设计)

2026-07-04 我们用 harness(462 因子 × 3 universe × 17 年 A 股 hfq,带 PIT 防幸存者偏差 + 训练集/样本外双重验证)**科学证伪**了"多因子轻松拿 alpha":

- 大盘(top300)/中盘(1200):全期有效因子在训练集(2021-23)全为 0 → 过拟合 2024-26 牛市
- 小盘(rank1500-3000):训练集 11 有效,但样本外(2024-26)**-24.5% / 回撤 -33.9%** → 直接崩

**因此本 SPEC 放弃"年化 50%"的虚高目标**,定位为**稳健可实盘型**:目标年化 10-20%、回撤 <20%;若回测落在 0-15% 或某年为负,那是 A 股近 5 年真相,非设计失败。

## 1. 因子选择(只用"训练集与全期方向一致"的因子)

07-04 IC 筛查里,仅这批因子**训练集(2021-23)与全期符号一致、未过拟合**:

| 因子 ID | 方向(高分=好) | 全期 ICIR | 训练集 ICIR | 性质 |
|---|---|---|---|---|
| fundamental_low_turnover | -turnover_rate(低换手) | 0.39 | 0.29 | 最干净,弱但稳 |
| fundamental_smb | -log(circ_mv)(小市值) | 0.10 | **0.27** | 训练集比全期强 |
| fundamental_bp | 1/pb(低市净) | 0.24 | 0.17 | 价值,弱但一致 |
| fundamental_sp | 1/ps_ttm(低市销) | 0.19 | 0.14 | 价值 |
| fundamental_ep | 1/pe_ttm(低市盈) | 0.11 | 0.14 | 价值 |
| fundamental_dp | dv_ttm(高股息) | 0.13 | 0.16 | 价值 |

**所有因子方向一致**:越便宜 / 越小盘 / 越低关注度 = 越高分 = 越该买。组合本质是 **"便宜 + 低关注 + 偏小盘" 的价值/逆向倾斜**——A 股少数经得起训练集考验的共性。

**显式排除**:alpha101_016 / gtja191_083 / qlib158_klow 等"全期有效但训练集 0"的过拟合因子,绝不入选。

## 2. 组合框架

- **Universe**: 中盘为主(`_midcap1200_mv.csv`,~1200 只)。**不碰纯小盘**(样本外已崩 -34%),中盘容量/流动性更实盘友好。
- **中性化**:`neutralize: true` → 市值(log_mcap)OLS 中性化(`neutralizer.py`,4/4 测试过),去除纯市值暴露。
- **加权**:等权(本版)。ICIR 加权列为可选调优(见 §6),等权最抗过拟合。
- **选股**:截面 Z-score + Top-N(`ZooSignalEngine` 已就绪)。本版 `top_n=30 / max_positions=30`,提升分散度。
- **调仓**:周频(周五信号→周一 next_open 换仓),守 `backtest-factory-no-trading-model-extension`(撮合时点归策略内部,工厂只给 next_open)。

## 3. 风控层(harness 6a 已建,直接启用)

| 参数 | 值 | 含义 |
|---|---|---|
| STOP_LOSS_THRESHOLD | -0.08 | 个股底线清仓 |
| EARLY_STOP_THRESHOLD | -0.05 | 持有≥3 日短期止损 |
| EARLY_KICK_THRESHOLD | +0.03 | 持有≥5 日仍不赚即淘汰 |
| 大盘择时 | MA20<MA60 | 熊市禁止买入 |

## 4. 成本与撮合(真实口径)

- 撮合 `next_open` + 涨跌停/停牌/双创幅度过滤(已就位)。
- 成本:slip 0.001 / commission 0.00025 / tax 0.0001(与 07-04 口径一致,可横向对比)。
- 注:本版为平铺成本;分层成本(高频因子换手侵蚀大)列为后续优化(见 §6)。

## 5. 验证方案(harness 全链路)

`multi_factor` 适配层 + `ic_screening.py` + `validation.py`(Monte Carlo/Walk-Forward)齐活。

- 因子有效性:**训练集 IC(2021-23)已先验确认**(§1 表格)。
- 策略表现:本回测跑 **2021-01-01 ~ 2026-06-30 全期**,年度拆分自然呈现 train(2021-23)与 OOS(2024-26)。
- PIT 防幸存者偏差已达标(astock 当日快照字段)。

## 6. 已知局限与后续(诚实留痕)

1. **行业中性化未启用**:`strategy.py` 当前只做 log_mcap 中性化(industry 硬编码 "ALL")。价值因子做行业内中性更优,需接入 `industry_map.csv` + 按日行业标签 → 后续增强。
2. **平铺成本**:未分层,高频换仓成本或被低估。
3. **静态 universe**:`_midcap1200_mv.csv` 为快照列表,非 PIT 逐日快照 → 轻微幸存者偏差。完整 PIT 需走 `pit_manifest`。
4. **ICIR 加权未用**:本版等权;若改用训练集 ICIR 加权需重跑确认不引入过拟合。
5. **容量/实盘**:中盘 Top30 容量中等;实盘需接 QMT 下单层(6+2 工程链路可复用)。

## 7. 诚实预期(预测)

- 乐观:年化 10-20%,回撤 <20%
- 现实:年化 0-15%,个别年份为负(尤其 2024 小盘杀跌外溢)
- 悲观:接近 0 或小幅为负(若价值因子衰减延续)

### 关键解读

1. **2021-2023 防御期反而跑赢**(累计 +18.7% 超额):便宜+小盘+低换手的价值倾斜在熊市是防御因子,符合因子逻辑。
2. **2024 单年 -42.4% 超额,直接归零**:2024 初小盘/微盘崩盘,本策略的"小盘+深度价值"暴露恰在最惨的位置,一年吞掉全部防御期超额还倒贴。
3. **净结果:比被动持有沪深300 还差 32%**。注意 universe 是静态快照(仅含存活到 2026 的票),存在幸存者偏差**向上**偏置——真实结果只会更差。

### 结论

- 即便最稳健、最经得起训练集考验的多因子设计,在 A 股 2021-2026 也**净亏损且跑输基准**。
- **年化 50% 的多因子目标,在分散框架下不仅"难",而是主动毁灭价值**。这把 2026-07-04 的证伪结论从"因子无 alpha"推进到"全期配置实盘即亏 32% 超额"的实锤。
- 要摸到 50% 年化,只能走**集中+杠杆+事件驱动**(用户已选"稳健可实盘型"放弃此路),且伴随 -40%+ 回撤、样本外脆弱——非稳健可实盘范畴。
